Installation of 3CX on Debian 12 Introduction This guide will help you to install 3CX on Debian 12. 3CX is a software-based private branch exchange (PBX) based on the SIP (Session Initiation Protocol) standard. It enables extensions to make calls via the public switched telephone network (PSTN) or via Voice over Internet Protocol (VoIP) services.
